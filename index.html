SFDA Inspection Preparation Checklist

For Low-Risk Disposable Medical Devices Factory (Based on MDS-REQ-010 & ISO 13485 Requirements)

1. Licenses & Authorizations

Item to CheckStatusNotes / Evidence
Valid SFDA manufacturing license available and current Done
Valid MDMA certificates for all products (check expiry) Done
Expired, damaged, or non-compliant devices isolated/disposed (records ready) Done

2. QMS Compliance

Item to CheckStatusNotes / Evidence
Valid ISO 13485 certificate (SFDA.MD/GSO ISO 13485:2017 or equivalent) from approved body Done
Quality Manual, SOPs, and internal audit records up-to-date and accessible Done
Records of previous SFDA inspections and corrective actions complete Done

3. Documentation

Item to CheckStatusNotes / Evidence
Technical files ready (device description, design/manufacturing info, risk management, validation/verification) Done
Procedures documented: traceability, UDI, storage/transport, complaints, recalls, destruction (retention 3-5+ years) Done
Labels, IFU, and advertising materials in Arabic/English and approved Done
All requested documents can be provided within 10 working days Done

4. Personnel

Item to CheckStatusNotes / Evidence
Sufficient qualified staff for all activities Done
Designated Authorized Representative to liaise with SFDA Done
Full-time Quality Manager & Technical Manager appointed with clear responsibilities Done

5. Facilities & Operations

Item to CheckStatusNotes / Evidence
Production and storage areas clean, suitable, and organized Done
Effective traceability and post-market surveillance procedures in place Done
All promotional/advertising materials SFDA-approved Done

6. Inspection Day Preparation

Item to CheckStatusNotes / Evidence
Dedicated room for inspectors (WiFi, printer, meeting space) Done
Translator available if needed Done
Safety gear and site access instructions prepared Done
Team briefed; ready for announced or unannounced visit Done
